# Escape from Hell

Survive the waves of enemies and escape from hell!

## Getting Started

### Prerequisites

- [Rust][1] (1.51.0 or later)

[1]: https://www.rust-lang.org/tools/install

### Running the Game

To run the game, run the following command:

```bash
cargo run --release
```
